Cómo importar familias de fuentes a través de link

Se establece el link con las distintas familias de fuentes a utilizar. Para ello se emplea el "&" para seleccionar más de una familia

<link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Lora&family=Roboto" rel="stylesheet">

Una vez indicado el link de las familias se ha de referenciar en la hoja de estilos CSS

h1 { font-family: 'Lora', serif; } p { font-family: 'Roboto', sans-serif; }

Ejemplo de título

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Dolores tempore cum rerum soluta? Voluptate voluptatibus maxime nulla, at officiis eius temporibus mollitia delectus. Necessitatibus expedita laborum quia beatae quae esse?

Cómo importar familias de fuentes a través de @import

Al igual que con link, "@import" importa las familias de fuentes y se referencia a través del CSS. No obstante, en este caso debe colocarse dentro de la hoja de estilos.

@import url('https://fonts.googleapis.com/css2?family=Fuzzy+Bubbles&family=Inconsolata');
