Iconos Font Awesome

Para utilizar los iconos de la librería Font Awesome bastará con registrarse en su sitio web para conseguir el código de usuario que nos permitirá establecer el link con dicha librería.
Para establecer el link debemos introducir el siguiente script:


<script src="https://kit.fontawesome.com/622ec138e7.js" crossorigin="anonymous"></script>

Ejemplo de iconos

<i class="fas fa-cloud"></i>
<i class="fas fa-heart"></i>
<i class="fas fa-car"></i>
<i class="fas fa-file"></i>
<i class="fas fa-bars"></i>

Aplicar tamaño y color a los iconos

<i class="fas fa-cloud" style="font-size:24px;"></i>
<i class="fas fa-cloud" style="font-size:36px;"></i>
<i class="fas fa-cloud" style="font-size:48px;color:red;"></i>
<i class="fas fa-cloud" style="font-size:60px;color:lightblue;"></i>
